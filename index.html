Completed Work

In December 2013, I completed the class Game Programming 1 where I worked with Stephen Sanders to produce a game over the course of a semester. The result is Rock 'Em Soccer Robots, which we made from scratch in C and C++. The game runs in Windows using DirectX 9.

I took the class Game Programming 2 in May 2014. The focus for this game was to use DirectX 11 and Bullet Physics. Like the previous game, it was produced from scratch in C and C++. While we did not have enough time to create actual enemies within the game (we simply made desks fly around, trying to kill you), our efforts resulted in a surprisingly fun experience.
